FastArray.php : 53

Colección indexada con API encadenable y contador de longitud explícito.

Mantiene un array interno y la propiedad $length sincronizada con las operaciones de inserción y eliminación. Implementa IteratorAggregate para foreach; los métodos públicos usan snake_case, salvo getIterator().

template
abstract

Debe extenderse para instanciar. Los métodos que devuelven static crean instancias de la subclase concreta.

note

item, first, last, pop y shift lanzan OutOfBoundsException con código 400 si el índice es inválido o la colección está vacía.

public splide(int $offset[, int|null $length = null ][, array<string|int, mixed>|mixed $replacement = [] ]) : static

Modifica $this->data in-place, recalcula $length con count() y devuelve los elementos eliminados en una nueva instancia de la subclase concreta.

Parameters
$offset : int

Índice de inicio (negativo cuenta desde el final).

$length : int|null = null

Cantidad a eliminar; null elimina hasta el final.

$replacement : array<string|int, mixed>|mixed = []

Valores de reemplazo; si no es array, se envuelve en uno.

Tags
example 
$array = new class([1, 2, 3, 4, 5]) extends FastArray };
$removed = $array->splide(1, 2, [8, 9]); // $array: [1, 8, 9, 4, 5]
