FastArray.php : 50

Colección de arrays con API orientada a métodos encadenables y seguimiento de longitud.

Implementa IteratorAggregate para permitir iteración con foreach. Los métodos públicos siguen la convención snake_case de DLStorage, salvo getIterator() requerido por la interfaz de PHP.

slice()

FastArray.php : 318

Devuelve una porción del array interno sin modificarlo.

public slice(int $offset[, int|null $length = null ][, bool $preserve_keys = false ]) : static

Utiliza array_slice para obtener un subarray a partir de $offset y con longitud $length. A diferencia de splide, no modifica el array interno, sino que devuelve una nueva instancia de FastArray que contiene únicamente los elementos seleccionados.

Parameters
$offset : int

Índice inicial para la porción. Puede ser negativo para contar desde el final del array.

$length : int|null = null

Número de elementos a incluir. Si es null, se seleccionan hasta el final del array.

$preserve_keys : bool = false

Indica si se deben conservar los índices originales. Por defecto es false.

Tags
example 
$array = new FastArray([10, 20, 30, 40, 50]);
$sub = $array->slice(1, 3);             // $sub contiene [20, 30, 40]
print_r($array->get());                  // Array original permanece [10, 20, 30, 40, 50]
$sub_preserved = $array->slice(1, 3, true); // Conserva índices: [1 => 20, 2 => 30, 3 => 40]
Return values
static —

Una nueva instancia de FastArray que contiene la porción seleccionada.

splide()

FastArray.php : 287

Modifica el array interno eliminando y/o reemplazando elementos, y devuelve los elementos eliminados.

public splide(int $offset[, int|null $length = null ][, array<string|int, mixed>|mixed $replacement = [] ]) : static

Este método utiliza array_splice para:

  1. Eliminar hasta $length elementos a partir del índice $offset.
  2. Reemplazar los elementos eliminados con los contenidos de $replacement.

La operación modifica directamente el array interno y reindexa los índices numéricos. Los elementos eliminados se devuelven como un nuevo objeto FastArray.

Parameters
$offset : int

Índice de inicio para la eliminación/reemplazo. Puede ser negativo para contar desde el final.

$length : int|null = null

Número de elementos a eliminar. Si es null, elimina hasta el final del array.

$replacement : array<string|int, mixed>|mixed = []

Elementos que reemplazarán a los eliminados. Si no es un array, se convierte automáticamente en uno.

Tags
example 
$array = new FastArray([1, 2, 3, 4, 5]);
$removed = $array->splide(1, 2, [8, 9]); // $array ahora: [1, 8, 9, 4, 5]
print_r($removed->get());               // [2, 3]
Return values
static —

Una nueva instancia de FastArray que contiene los elementos eliminados.
