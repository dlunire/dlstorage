Codificación y decodificación de datos mediante transformación con entropía.

Cada byte de entrada se convierte en un bloque hexadecimal de 10 caracteres (BinaryLengthTrait::to_hex40()), opcionalmente alterado por una llave de entropía. Los bloques se concatenan y forman el payload de un archivo .dlstorage.

get_decode()

Data.php : 97

Decodifica una cadena previamente codificada con {@see encode()}.

public get_decode(string $encoded[, string|null $entropy = null ]) : string
  1. Restaura ceros compactados con expand_zero().
  2. Divide en bloques de 10 caracteres hex.
  3. Revierte la entropía con get_reverse_entropy().
  4. Valida que la longitud UTF-8 del resultado sea par; si no, lanza excepción.
Parameters
$encoded : string

Cadena hexadecimal producida por encode().

$entropy : string|null = null

Misma llave usada al codificar.

Tags
throws
EncodeException

Si la longitud resultante es impar (entropía incorrecta o datos corruptos, código 403).

Return values
string —

Cadena de bytes originales (caracteres de un solo byte).

public get_entropy_value(string $input) : int

Suma todos los bytes con array_sum(unpack("C*", ...)), deriva $this->coefficient mediante calculate_coefficient() y retorna suma + strlen($input).

No es entropía de Shannon; es una heurística interna del MTB.
