Storage.php : 66

Interfaz pública para lectura y escritura de archivos en formato binario DLStorage.

Punto de entrada recomendado de la biblioteca. Extiende SaveData y encapsula la ruta del archivo y la entropía en el constructor, delegando la persistencia en generate() y la recuperación en readfile().

Responsabilidades:

  • Normalizar separadores de ruta (/ → DIRECTORY_SEPARATOR) durante la construcción.
  • Generar archivos .dlstorage mediante generate().
  • Leer y decodificar archivos mediante readfile().
  • Exponer la firma y versión binaria del formato mediante get_current_signature() y get_current_version().

Jerarquía de herencia: Storage → SaveData → DataStorage → Data.

read_storage_data()

SaveData.php : 122

Lee un archivo `.dlstorage` y devuelve el contenido original decodificado.

public read_storage_data(string $filename[, string|null $entropy = null ][, bool $storage = true ]) : string

Flujo de lectura (offsets 1-based):

  • Bytes 1–9: firma (DLStorage).
  • Bytes 10–13: tamaño de la sección de versión (entero, 4 bytes).
  • Bytes 14..(14+header_size-1): versión.
  • 4 bytes siguientes: tamaño del payload.
  • Bytes restantes: payload hexadecimal, decodificado con Data::get_content().
Parameters
$filename : string

Nombre base; se añade .dlstorage automáticamente.

$entropy : string|null = null

Llave usada al codificar. Debe coincidir con la de escritura.

$storage : bool = true

true: busca en storage/. false: busca en la raíz del proyecto.

Tags
throws
StorageException

Archivo inexistente (404), firma inválida (500) o error de lectura.

Return values
string —

Contenido original tras decodificación.

save_data()

SaveData.php : 73

Codifica y persiste datos en un archivo `.dlstorage`.

public save_data(string $filename, string $data[, string|null $entropy = null ][, bool $storage = true ]) : void

Flujo:

  1. Data::encode() transforma $data en una cadena hexadecimal.
  2. Se arma una cadena hex con: firma + tamaño(versión) + versión + tamaño(payload) + payload.
  3. hex2bin() convierte la cadena completa a binario y se escribe en disco.
  4. Si $storage es true, la ruta se resuelve bajo {raíz}/storage/.
